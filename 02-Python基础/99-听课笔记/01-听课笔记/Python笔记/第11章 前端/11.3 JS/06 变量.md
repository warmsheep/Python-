## 变量

变量是用来存储信息的容器，由于JS是松散数据类型，所以变量中可以放置任何的数据类型。创建变量 储存值和代表值。

### 第一种方法:先声明再定义
声明 创建，此时这个变量中什么都没有，获取变量中的返回值时，返回值为undefind

```javascript
var a;
alert(dog)
dog="小黄"
alert(dog);
```

= 在JS中是赋值的意义将等号右侧的值赋值给等号左边的变量

+= 在JS中是保留原来的值上累加

一次声明多个变量，

JS中代码的执行顺序:自上而下

### 第二种方法:声明时立刻定义

```javascript
var dog2="小红";
alert(dog)
```
