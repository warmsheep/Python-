## 字体样式

### 字体样式处理
##### font-family
设置字体类型

```html
font-family:"隶书";
```

##### font-size
设置字体大小

```html
font-size:12px;
```
##### font-style
设置字体风格
* 正常:normal
* 斜体:italic,oblique

```html
font-style:italic;
```
##### font-weight
* normal | bold | bolder | lighter | <interger>

normal:400
bold:700
interger用数字表示文本字体的粗细，取值范围: 100|200|300|400|500|600|700|800|900

设置字体的粗细

```html
font-weight:bold;
```

##### font
在一个声明中设置所有字体属性

```html
font:italic bold 36px "宋体";
```
