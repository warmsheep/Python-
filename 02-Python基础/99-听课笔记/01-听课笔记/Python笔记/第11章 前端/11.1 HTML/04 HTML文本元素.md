## HTML文本元素

### HTML基本元素标签

| 名称 | 标签 |
|:-- |:-- |
| 标题标签 | \<h1><\/h1>~\<h6><\/h6> |
| 段落 | \<p><\/p>  |
| 换行符 | <br\/\> |
| 水平线 | <hr\/\> |
| 斜体 | \<i><\/i>、\<em><\/em> |
| 字体加粗 | \<strong><\/strong> |

* 标题标签是块级标签，自成一行


##### head标签

* 编码信息
```html
<meta charset="utf-8">
```

* title信息
```html
<title></title>
```

##### html标签

* 设置语言
```html
<html lang="en">
```

##### body标签
主要放浏览器存储的内容





### HTML注释
```
<!-- -->
```

### HTML特殊符号

| 特殊符号 | 字符实体 |
|:-- |:-- |
| 空格 | \&nbsp\; |
| > | \&gt\; |
| < | \&lt\; |
| * | \&quot\; |
| @ | \&copy\; |
