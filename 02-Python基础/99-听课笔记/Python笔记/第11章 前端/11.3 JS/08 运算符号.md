## 运算符号

### 运算符号分类
* 算术运算
* 赋值运算
* 比较运算符
* 逻辑运算符

### 算术运算
| 运算符 | 描述 |
|:-- |:-- |
| + | 加 |
| - | 减 |
| * | 乘 |
| / | 除 |
| % | 求余数(保留整数) |
| ++ | 累加 |
| -- | 递减 |

### 赋值运算

| 运算符 | 描述 |
|:-- |:-- |
| = | x=y |
| += | x+=y |
| *= | x*=y |
| /= | x/=y |
| %= | x%=y |

### 比较运算符

| 运算符 | 描述 |
|:-- |:-- |
| == | 等于 |
| === | 全等 |
| != | 不等于 |
| > | 大于 |
| < | 小于 |
| >= | 大于或等于 |
| <= | 小于或等于 |

```javascript
//隐式转换==比较的是值得大小，===是类型和值都进行比较
console.log("5"==5)//true
console.log("5"===5)//false
```

### 逻辑运算符

| 运算符 | 描述 |
|:-- |:-- |
| && | and |
| \|\| | or |
| ! | not |
