## 超链接

跳转到页面不同的地方，或者跳转到不同的页面，或者跳转到不同的网站

### 跳转到页面不同位置

对需要跳转的标签设置id="sth"
然后用超链接，记住在id的内容前面需要加上#
```html
<a herf="#sth">跳转到某标签</a>
```

### 跳转到不同的页面
```html
<a herf="./列表.html">列表笔记</a>
```

### 跳转到不同的网站

* 直接在当前页面跳转
```html
<a href="https://www.baidu.com">百度一下</a>
```

* 重新开启浏览窗口跳转
```html
<a href="https://www.baidu.com" target="_blank">百度一下</a>
```
