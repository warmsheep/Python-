## 列表

列表分为有序和无序

### 有序列表

```html
<ol></ol>
列表里面的元素用<li></li>
```

举个例子
```html
<ol>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
</ol>
```


输出结果

```html
1.1
2.2
3.3
4.4
```



### 无序列表

```html
<ul></ul>
列表里面的元素用<li></li>
```

举个例子

```html
<ul>
  <li>1</li>
  <li>2</li>
  <li>3</li>
  <li>4</li>
</ul>
```


输出结果

```html
.1
.2
.3
.4
```

在开发中一般使用无序列表
