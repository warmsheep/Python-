# 读写模式

## 读写模式
* 打开模式只有只读、只写、只追加，难道没有可以读写的操作么？当然有

**示例七：读写模式**

```python
f = open("/Users/yuanjun/Python-Note/02-Python基础/99-听课笔记/代码案例文稿/通讯录.txt",'r+',encoding="utf-8")

data=f.read() #可以读内容
f.write("范冰冰 女 11122522378 山东 青岛 汉族"+"\n")
f.close()
```

**上面的内容加到哪里了呢？**
* 答案是追加到了最后面。

**那如果是我想添加到任意位置呢？**
* 答案是可以，后面会讲到
