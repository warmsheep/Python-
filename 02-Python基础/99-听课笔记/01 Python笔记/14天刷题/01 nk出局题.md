### 题目

josephus_survivor（7,3）=>表示7个人在一个圈子里;
每3人中有一人被淘汰，直到剩下一人
[1,2,3,4,5,6,7] - 初始序列
[1,2,4,5,6,7] => 3被计数
[1,2,4,5,7] => 6被计数
计算出[1,4,5,7] => 2
计算出[1,4,5] => 7
计数[1,4] => 5
[4] => 1计数，4是最后一个元素 - 幸存者！

现在给任意的n,k值，josephus_survivor（n,k)，求幸存者。
